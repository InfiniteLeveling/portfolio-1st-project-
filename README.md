# 🌐 Gouranga . Dev — Personal Portfolio

A sleek, modern, and high-performance personal developer portfolio built using HTML5 and custom CSS3. Designed with a futuristic dark theme, glowing gradient accents, glassmorphism UI elements, and smooth micro-animations.

---

## ✨ Features

- **🎨 Futuristic Design System**: Built with deep dark backgrounds (`#030008` to `#14082e`) paired with vibrant neon violet & blue glowing gradients.
- **🔍 Glassmorphism Navigation**: Sticky, semi-transparent navbar with `backdrop-filter: blur(10px)` and interactive glowing hover indicator lines.
- **🚀 Hero Section**: Catchy introduction headline, biography preview, and prominent Call-to-Action (CTA) buttons ("Enter project" & "Let's talk").
- **🖼️ Interactive Avatar**: Circular glowing profile frame with smooth multi-layer glow box-shadow effects and scale transitions on hover.
- **📱 Clean & Modular Codebase**: Pure semantic HTML5 and vanilla CSS3 without heavy third-party framework dependencies.

---

## 🛠️ Tech Stack

- **HTML5**: Semantic web architecture (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`).
- **CSS3**: Custom styling, CSS Flexbox layout, keyframe styling, CSS filters (`backdrop-filter`), and CSS gradient text clips.
- **Typography**: Google Sans, Merriweather, and Roboto.

## 🚀 Quick Start

1. **Clone the Repository**:
   ```bash

   ```

2. **Open the Project**:
   - Open `index.html` directly in any web browser.
   - Or run with VS Code **Live Server** extension for live reloading.

---

## 🔧 Customization

To personalize this portfolio with your details:

1. **Name & Text**: Update your name, title, and bio in `index.html`.
2. **Profile Image**: Replace `assets/Gemini_Generated_Image_y5kwhzy5kwhzy5kw.png` with your image or update the path in `style.css` under `.profile-pic`.
3. **Links & Navigation**: Connect the navigation anchor links (`#about`, `#skills`, `#projects`, `#contact`) to your respective content sections or pages.
